## Prereqs to install
1. [node.js](https://nodejs.org/)
2. [express](https://expressjs.com/)

## How to run
1. Go to the root folder of the project
2. Execute `node server.js` in the terminal
3. Open browser to access `localhost:5000`
4. Enjoy
